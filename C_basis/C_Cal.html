printf 문

연산자는 크게 대입 연산자와 산술 연산자로 나눌 수 있다.
대입 연산자는 '=' 으로 대표 되며,
산술 연산자는 (+, - , * , /, %) 등으로 설명 가능하다.

예제)
#include< stdio.h >
  int main(){
    int a=3;
      a=a*3;
      printf("%d \n",a);
      a=a-3;
      printf("%d \n",a);
      a=a/3;
      printf("%d \n",a);
      a=a%3;
      printf("%d \n",a);
      return 0;
  }

이 코드를 실행 하면,
정수 a는 3에서 대입 연산자로 우측에 있는 산술 연산자에 의해
곱해지고'*' 빼지고'-' 나눠지고'/' 나눈 값의 나머지'%' 를
printf에 의해 출력 된다.
