printf 문

이 예제는 처음 코딩을 배울 때 자주 쓰이는 기초적인 예제이다.

예제)
#include< stdio.h >
  int main(){

  printf("Hello World!");

}

이 코드를 실행 하면,
printf 에 의해 (괄호)속의 "큰 따옴표"에 있는 문장인
Hello World! 가 출력 된다.
printf()는 c언어에서 아주 기초적으로 자주 쓰이는 함수이다.
잘 숙지해 두는 것이 좋다.
