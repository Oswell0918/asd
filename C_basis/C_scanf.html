scanf 문

이 예제는 printf 와 비슷한 기초적인 예제이다.

예제)
#include< stdio.h >
  int main(){
    int a; // a는 정수입니다.
    printf("숫자를 입력하세요:"); // "큰 따톰표" 안의 문장을 출력
    scanf("%d",&a); // 정수 a값을 입력
    printf("%d",a); // 입력 받은 정수 a값을 츨력.
  }

이 코드를 실행 하면,
printf 에 의해 (괄호)속의 "큰 따옴표"에 있는 문장인
숫자를 입력하세요: 뒤에 scanf로 정수 a값에 숫자를 적고 enter 키를 눌러 입력할 수 있다.
입력 된 정수는 다시 출력 된다.
