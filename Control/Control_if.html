if, else if, else 문

if문은 대표적인 제어문이다.

예문)
#include< stdio.h >
int main(){

  int a;
  scanf("%d",&a);
  if(a==1){
    printf("a는 1이다!");
  }
  else{
    printf("a는 1이 아니다!");
  }
}

이 프로그램을 돌리면 a라는 정수에 정수값을 입력하고
if문으로 a값이 1이면 "a는 1이다!"
a가 1이 아니면 else의 "a는 1이 아니다" 가 출력 된다.
