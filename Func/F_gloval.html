전역 변수 / 지역 변수

지역 변수는 해당 함수내에서만 유효한 변수이며,
전역 변수는 프로그램 전체영역에서 접근 가능한 변수이다.

#include< stdio.h > int a=10; // 전역 변수
int main(){
  int num, b=2; //지역 변수
  scanf("%d",&num);
  printf("%d \n %d \n %d \n",a, num, b);
}

전역 변수인 a는 main 함수내에서 선언 되지 않았음에도 printf에서 출력 됐다
지역 변수인 b와 num은 main함수안에서 선언 되어서 출력됐다.
