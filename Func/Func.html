재귀함수

함수는

예제)
#include< stdio.h >
int comp(int a, int b){
  if(a>b) return a;
  else return b;
}
main(){
  int a, b;
  scanf("%d %d", &a,&b);
  printf("%d \n", comp(a,b));
}

이 코드를 실행 하면,
int main 속에서 정의 된 정수 a와b가
scanf로 값이 입력 되고
printf에서 comp함수가 실행 되는데 이 comp함수는
main에서 입력받은 두 정수 a와 b를 비교해준다
comp함수 속 코드는 if , else문으로
만약 a가 b보다 크다면 a를 반대라면 b를
저장하게 한다.
고로 a 와 b중 큰 값을 출력 받게 된다.
